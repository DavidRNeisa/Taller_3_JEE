Clase 23: Tablas de Dispersión (Hash Tables)

Acceso en O(1): cómo funcionan los mapas y diccionarios

Concepto fundamental

Una tabla hash almacena pares clave-valor. Una función hash transforma la clave en un índice del arreglo, permitiendo acceso en O(1) promedio.

Analogía: Un casillero con cerradura. La clave es tu nombre; la función hash es la regla que asigna un número de casillero a cada persona. Accedes directamente a tu casillero sin buscar en todos.

Función hash

// Hash simple por módulo
int hash(int clave, int tamanio) {
    return clave % tamanio;
}

// Hash para strings (djb2)
unsigned long hashStr(const string& s) {
    unsigned long h = 5381;
    for (char c : s) h = ((h << 5) + h) + c;  // h*33 + c
    return h;
}
// Propiedades deseables: distribución uniforme, rápida de calcular

Colisiones

Dos claves distintas pueden producir el mismo índice. Hay dos estrategias principales para resolverlas:

1. Encadenamiento (Chaining)

// Cada celda del arreglo tiene una lista enlazada
vector<list<pair<int,int>>> tabla(10);

void insertar(int clave, int valor) {
    int idx = hash(clave, tabla.size());
    // Buscar si ya existe y actualizar
    for (auto& [k,v] : tabla[idx])
        if (k==clave) { v=valor; return; }
    tabla[idx].push_back({clave, valor});
}

int buscar(int clave) {
    int idx = hash(clave, tabla.size());
    for (auto& [k,v] : tabla[idx])
        if (k==clave) return v;
    return -1;  // no encontrado
}
// Complejidad promedio O(1); peor caso O(n) si todo colisiona

2. Direccionamiento Abierto — Sondeo Lineal

// Todos los elementos en el arreglo; si hay colisión, buscar la siguiente celda libre
int tabla[TAMANIO];
fill(tabla, tabla+TAMANIO, -1);  // -1 = vacío

void insertar(int clave) {
    int idx = hash(clave, TAMANIO);
    while (tabla[idx] != -1)
        idx = (idx+1) % TAMANIO;  // avanzar linealmente
    tabla[idx] = clave;
}
// Problema: clustering primario — cadenas de celdas ocupadas

La STL: unordered_map

#include <unordered_map>
unordered_map<string,int> mapa;
mapa["Ana"]  = 95;
mapa["Luis"] = 87;
cout << mapa["Ana"];        // 95
mapa.erase("Luis");          // eliminar
cout << mapa.count("Luis"); // 0 — no existe
