Clase 20: Árboles de Codificación — Huffman

Compresión sin pérdida mediante códigos de longitud variable

El problema de la compresión

En ASCII, cada carácter ocupa 8 bits, sin importar su frecuencia. Si en un texto 'e' aparece 1000 veces y 'z' aparece 2 veces, ambos usan 8 bits. La idea de Huffman: asignar menos bits a los caracteres más frecuentes.

Algoritmo de Huffman

  1. Calcular la frecuencia de cada símbolo.
  2. Crear un nodo hoja por cada símbolo.
  3. Usar un min-heap con los nodos ordenados por frecuencia.
  4. Repetir hasta que quede un solo nodo: extraer los dos mínimos, crear un nodo padre con frecuencia = suma de ambos, insertarlo en el heap.
  5. El árbol resultante define los códigos: 0 para ir a la izquierda, 1 para la derecha.
// Ejemplo: texto "aabbbccdddd"
// Frecuencias: a=2, b=3, c=2, d=4
// Total: 11 caracteres

// Paso 1: heap = [(a,2),(c,2),(b,3),(d,4)]
// Paso 2: unir a y c → nodo(4), heap = [(b,3),(d,4),(ac,4)]
// Paso 3: unir b y nodo(ac,4) → nodo(7), heap = [(d,4),(bac,7)]
// Paso 4: unir d y nodo(bac,7) → raíz(11)

//           11
//          /   \
//         4     7
//        (d)   / \
//             3   4
//            (b)  / \
//                2   2
//               (a) (c)
// Códigos: d=0, b=10, a=110, c=111

Implementación en C++

struct NodoHuff {
    char c; int freq;
    NodoHuff *izq, *der;
    bool operator>(const NodoHuff& o) const { return freq > o.freq; }
};

void generarCodigos(NodoHuff* r, string codigo, map<char,string>& tabla) {
    if (!r) return;
    if (!r->izq && !r->der) tabla[r->c] = codigo;
    generarCodigos(r->izq, codigo+"0", tabla);
    generarCodigos(r->der, codigo+"1", tabla);
}

NodoHuff* construirArbol(map<char,int>& freq) {
    priority_queue<NodoHuff*, vector<NodoHuff*>,
                   function<bool(NodoHuff*,NodoHuff*)>>
        pq([](NodoHuff* a, NodoHuff* b){ return a->freq > b->freq; });

    for (auto& [c,f] : freq) pq.push(new NodoHuff{c,f,nullptr,nullptr});

    while (pq.size() > 1) {
        auto l = pq.top(); pq.pop();
        auto r = pq.top(); pq.pop();
        pq.push(new NodoHuff{0, l->freq+r->freq, l, r});
    }
    return pq.top();
}
