Clase 14: Árboles Binarios de Búsqueda (BST)

La estructura para búsqueda dinámica eficiente

Propiedad BST

En un BST, para cada nodo n:

  • Todo valor en el subárbol izquierdo es menor que n.
  • Todo valor en el subárbol derecho es mayor que n.

Esta propiedad permite búsqueda binaria eficiente dentro del árbol.

Inserción

struct Nodo { int v; Nodo *izq,*der; Nodo(int x):v(x),izq(nullptr),der(nullptr){} };

Nodo* insertar(Nodo* raiz, int val) {
    if (!raiz) return new Nodo(val);
    if (val < raiz->v) raiz->izq = insertar(raiz->izq, val);
    else if (val > raiz->v) raiz->der = insertar(raiz->der, val);
    // si val == raiz->v: ignorar duplicado
    return raiz;
}
// Traza con valores 10,5,15,3,7:
//        10
//       /  \
//      5   15
//     / \
//    3   7

Búsqueda

bool buscar(Nodo* raiz, int val) {
    if (!raiz) return false;
    if (val == raiz->v) return true;
    if (val < raiz->v) return buscar(raiz->izq, val);
    return buscar(raiz->der, val);
}
// En cada paso descartamos la mitad del árbol → O(log n) promedio

Eliminación — el caso más complejo

Nodo* minNodo(Nodo* n) {
    while (n->izq) n = n->izq;
    return n;
}
Nodo* eliminar(Nodo* raiz, int val) {
    if (!raiz) return nullptr;
    if (val < raiz->v) raiz->izq = eliminar(raiz->izq, val);
    else if (val > raiz->v) raiz->der = eliminar(raiz->der, val);
    else {
        // Caso 1: hoja
        if (!raiz->izq && !raiz->der) { delete raiz; return nullptr; }
        // Caso 2: un solo hijo
        if (!raiz->izq) { Nodo* t=raiz->der; delete raiz; return t; }
        if (!raiz->der) { Nodo* t=raiz->izq; delete raiz; return t; }
        // Caso 3: dos hijos → reemplazar con sucesor inorden
        Nodo* suc = minNodo(raiz->der);
        raiz->v = suc->v;
        raiz->der = eliminar(raiz->der, suc->v);
    }
    return raiz;
}
