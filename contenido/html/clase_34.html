Clase 34: Parcial 3 — Guía de Preparación

Todo lo que debes dominar sobre árboles y grafos para la evaluación

Temario del Parcial 3

  • Árboles BST: inserción, búsqueda, eliminación, recorridos.
  • Árboles AVL: factor de balance, tipos de rotación.
  • Heaps: inserción, extracción, heapify.
  • Tablas hash: función hash, colisiones, factor de carga.
  • Grafos: DFS, BFS, Dijkstra, Floyd-Warshall, MST.

Ejercicios de repaso rápido

Árbol BST

// Insertar: 50, 30, 70, 20, 40, 60, 80
//           50
//          /   \
//        30     70
//       /  \   /  \
//      20  40  60  80
// Inorden: 20 30 40 50 60 70 80 ✓
// Altura: 3. Balance de raíz: 0.
// Eliminar 30 (dos hijos):
//   Sucesor inorden = 40. Reemplazar 30 por 40, eliminar 40 de su posición.

Heap

// Max-Heap: insertar 10, 20, 15, 30, 40
// Paso a paso: [10] → [20,10] → [20,10,15] → [30,20,15,10] → [40,30,15,10,20]
//      40
//     /   \
//    30    15
//   /  \
//  10   20
// Extraer máximo (40): poner 20 en raíz, heapify-down
// Resultado: [30,20,15,10]

Dijkstra — traza
