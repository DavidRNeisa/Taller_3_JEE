Clase 17: Montículos (Heaps)

Árbol binario completo con orden parcial: la cola de prioridad eficiente

Definición

Un heap es un árbol binario completo que satisface la propiedad de heap:

  • Max-Heap: el valor de cada nodo es ≥ el de sus hijos. La raíz es el máximo.
  • Min-Heap: el valor de cada nodo es ≤ el de sus hijos. La raíz es el mínimo.

Representación en arreglo

Al ser un árbol completo, se puede representar eficientemente en un arreglo sin punteros:

// Para el nodo en índice i:
int padre(int i)    { return (i-1)/2; }
int hijoIzq(int i)  { return 2*i+1; }
int hijoDer(int i)  { return 2*i+2; }

// Max-Heap: [90, 80, 70, 40, 50, 60, 30]
//              90
//            /    \
//          80      70
//         /  \    /  \
//        40  50  60   30

Operación Heapify-Up (inserción)

void push(vector<int>& heap, int val) {
    heap.push_back(val);
    int i = heap.size()-1;
    // subir hasta que el padre sea mayor o sea la raíz
    while (i > 0 && heap[padre(i)] < heap[i]) {
        swap(heap[i], heap[padre(i)]);
        i = padre(i);
    }
}
// Complejidad: O(log n)

Operación Heapify-Down (extracción del máximo)

int pop(vector<int>& heap) {
    int maximo = heap[0];
    heap[0] = heap.back();
    heap.pop_back();
    int i = 0, n = heap.size();
    while (true) {
        int mayor = i;
        int l = hijoIzq(i), r = hijoDer(i);
        if (l < n && heap[l] > heap[mayor]) mayor = l;
        if (r < n && heap[r] > heap[mayor]) mayor = r;
        if (mayor == i) break;
        swap(heap[i], heap[mayor]);
        i = mayor;
    }
    return maximo;
}
// Complejidad: O(log n)

Construcción en O(n) — Build-Heap

// Aplicar heapify-down desde el último nodo interno hacia la raíz
void buildHeap(vector<int>& arr) {
    for (int i = arr.size()/2 - 1; i >= 0; i--)
        heapifyDown(arr, i);
}
// Aunque parece O(n log n), el análisis amortizado da O(n)

Heapsort: ordenamiento en O(n log n)

// 1. Build-Heap: O(n)
// 2. n veces: extraer máximo y poner al final: O(n log n)
// In-place, sin memoria extra — pero no estable

STL: priority_queue

#include <queue>
priority_queue<int> maxH;           // max-heap por defecto
priority_queue<int,vector<int>,greater<int>> minH; // min-heap
maxH.push(5); maxH.push(1); maxH.push(8);
cout << maxH.top(); // 8
maxH.pop();
