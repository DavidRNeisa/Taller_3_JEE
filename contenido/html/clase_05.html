Clase 5: Templates y Programación Genérica en C++

Escribir código una vez, usarlo con cualquier tipo de dato

El problema sin templates

Sin templates, necesitas escribir versiones separadas de una función para cada tipo:

int    maxInt   (int a, int b)       { return a>b ? a : b; }
double maxDouble(double a, double b) { return a>b ? a : b; }
// Código duplicado — viola el principio DRY (Don't Repeat Yourself)

Templates de función

template <typename T>
T maximo(T a, T b) {
    return a > b ? a : b;
}

cout << maximo<int>(3, 7);         // 7
cout << maximo<double>(3.5, 2.1); // 3.5
cout << maximo('z', 'a');          // 'z' — tipo deducido automáticamente

Templates de clase: pila genérica

template <typename T>
class Pila {
    T    datos[100];
    int  cima = -1;
public:
    void push(T val) {
        if (cima < 99) datos[++cima] = val;
    }
    T pop() {
        if (cima < 0) throw runtime_error("Pila vacía");
        return datos[cima--];
    }
    T    top()       const { return datos[cima]; }
    bool estaVacia() const { return cima == -1; }
};

Pila<int>    pilaInt;
Pila<string> pilaStr;
pilaInt.push(42);
pilaStr.push("Hola");

Template con múltiples parámetros de tipo

template <typename K, typename V>
class Par {
public:
    K clave; V valor;
    Par(K k, V v) : clave(k), valor(v) {}
};

Par<string, int> nota("Ana", 95);
cout << nota.clave << ": " << nota.valor;

Template con parámetro no-tipo

template <typename T, int N>
class ArregloFijo {
    T datos[N];
public:
    T&  operator[](int i) { return datos[i]; }
    int tamanio()   const { return N; }
};

ArregloFijo<int, 10> arr;
arr[0] = 5;
cout << arr.tamanio(); // 10

Ventajas vs. consideraciones

VentajaConsideración
Reutilización total del códigoErrores de tipo a veces son mensajes crípticos
Sin overhead en tiempo de ejecuciónPuede aumentar el tamaño del binario
Seguridad de tipos garantizadaEl código del template debe estar en el header
La STL entera está basada en templates. Cuando escribes vector<int>, el compilador instancia una versión especializada de vector para enteros en tiempo de compilación.
