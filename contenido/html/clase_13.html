Clase 13: Introducción a Árboles

Estructuras jerárquicas: terminología, propiedades y representación

¿Qué es un árbol?

Un árbol es una estructura de datos jerárquica compuesta por nodos conectados por aristas. Es un grafo conexo acíclico: no hay ciclos y existe exactamente un camino entre cualquier par de nodos.

Terminología fundamental

TérminoDefinición
RaízNodo superior sin padre
HijoNodo directamente conectado debajo de otro
PadreNodo directamente conectado encima de otro
HojaNodo sin hijos
AlturaLongitud del camino más largo desde la raíz hasta una hoja
Profundidad de nodoLongitud del camino desde la raíz hasta ese nodo
SubárbolUn nodo y todos sus descendientes

Ejemplo visual

        A          ← raíz (profundidad 0)
       / \
      B   C        ← profundidad 1
     / \   \
    D   E   F      ← hojas (profundidad 2)

Altura del árbol: 2
Hojas: D, E, F
Padre de B: A
Hijos de B: D, E

Árbol binario

En un árbol binario cada nodo tiene como máximo dos hijos (izquierdo y derecho).

struct Nodo {
    int   valor;
    Nodo* izq;
    Nodo* der;
    Nodo(int v) : valor(v), izq(nullptr), der(nullptr) {}
};

Propiedades importantes de árboles binarios

  • Un árbol binario con altura h tiene como máximo 2^(h+1) - 1 nodos.
  • Un árbol binario completo tiene todos los niveles llenos excepto posiblemente el último (que se llena de izquierda a derecha).
  • Un árbol binario perfecto tiene todos los niveles completamente llenos.

Recorridos básicos

// Inorden (Izq → Raíz → Der): produce valores en orden ascendente en BST
void inorden(Nodo* r) {
    if (!r) return;
    inorden(r->izq); cout << r->valor << " "; inorden(r->der);
}
// Preorden (Raíz → Izq → Der): útil para copiar el árbol
void preorden(Nodo* r) {
    if (!r) return;
    cout << r->valor << " "; preorden(r->izq); preorden(r->der);
}
// Postorden (Izq → Der → Raíz): útil para eliminar el árbol
void postorden(Nodo* r) {
    if (!r) return;
    postorden(r->izq); postorden(r->der); cout << r->valor << " ";
}

Aplicaciones

  • Sistemas de archivos: carpetas y subcarpetas forman un árbol.
  • DOM en HTML: estructura jerárquica de una página web.
  • Jerarquías organizacionales: CEO → directores → gerentes → empleados.
  • Compiladores: árbol de sintaxis abstracta (AST).
