Clase 4: Taller de Complejidad Algorítmica

Análisis práctico: determinar Big-O paso a paso

Metodología de análisis

  1. Identificar las operaciones elementales (comparaciones, asignaciones).
  2. Contar cuántas veces ejecuta cada estructura de control.
  3. Expresar el total en función de n.
  4. Conservar el término dominante, eliminar constantes.

Ejercicio 1: Bucles anidados independientes

for (int i = 0; i < n; i++)        // n veces
    for (int j = 0; j < n; j++)    // n veces por i
        cout << i + j;             // O(1)
// Total: n × n = n² → O(n²)

Ejercicio 2: Bucle con límite dependiente de i

for (int i = 0; i < n; i++)
    for (int j = 0; j < i; j++)   // 0+1+2+...+(n-1)
        operacion();
// Total: n(n-1)/2 → O(n²)

Ejercicio 3: Búsqueda lineal — análisis por caso

int buscar(int arr[], int n, int x) {
    for (int i = 0; i < n; i++)
        if (arr[i] == x) return i;
    return -1;
}
// Mejor caso: O(1) — encontrado en primera posición
// Peor caso:  O(n) — no está, o está al final
// Promedio:   O(n/2) → O(n)

Ejercicio 4: División recursiva

void dividir(int n) {
    while (n > 1) {
        cout << n;
        n = n / 2;   // n se divide entre 2 en cada paso
    }
}
// Iteraciones = log₂(n) → O(log n)

Ejercicio 5: Bucles secuenciales

void combinado(int n) {
    for (int i = 0; i < n; i++)           // O(n)
        cout << i;
    for (int i = 0; i < n; i++)           // O(n²)
        for (int j = 0; j < n; j++)
            cout << i*j;
}
// O(n) + O(n²) = O(n²) — domina el mayor

Ejercicio 6: Bucle externo logarítmico

void misterio(int n) {
    for (int i = 1; i <= n; i *= 2)   // O(log n) iteraciones
        for (int j = 0; j < n; j++)   // O(n) cada vez
            cout << i + j;
}
// Total: O(n log n)
